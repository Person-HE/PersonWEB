## 1. 产品概述

个人工具与资源导航站——一个纯前端的个人品牌网站，聚合免费小工具入口、精选资源下载、个人名片和网站导航四大板块。用户无需注册登录，打开即用；站长可通过内置管理面板动态添加工具、资源和收录网站。

- 目标用户：寻找实用工具和资源的互联网用户，以及对站长个人品牌感兴趣的人群
- 核心价值：一站式获取优质工具与资源，零门槛使用，持续更新

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览、搜索、分类筛选、复制网盘链接 |
| 站长 | 本地密码验证 | 添加/编辑/删除工具、资源、网站导航 |

### 2.2 功能模块

1. **首页**：Hero区域展示站名与标语，四大模块入口卡片，最新更新推荐
2. **工具集页面**：工具卡片列表，搜索栏，分类标签筛选，工具详情弹窗（描述、配图、网盘链接可复制）
3. **资源下载站页面**：资源卡片列表，搜索栏，分类标签筛选，资源详情弹窗（描述、配图、网盘链接可复制）
4. **网站导航页面**：网站卡片列表，搜索栏，分类标签筛选，点击直达目标网站
5. **个人名片页面**：个人故事介绍，做站初衷，微信二维码入口
6. **管理面板**：添加/编辑/删除工具、资源、网站，密码保护入口

## 3. 核心流程

### 访客浏览流程
用户打开网站 → 首页展示四大模块入口 → 点击进入对应模块 → 搜索/分类筛选找到目标 → 查看详情 → 复制网盘链接/跳转网站

### 站长管理流程
站长点击管理入口 → 输入密码验证 → 进入管理面板 → 选择模块（工具/资源/网站） → 填写表单添加新条目 → 数据保存至localStorage

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色系（#0F172A 深蓝黑）+ 亮色强调（#06D6A0 翡翠绿）
- **辅助色**：#FFD166 暖黄（高亮）、#EF476F 珊瑚红（标签）
- **按钮风格**：圆角微凸（rounded-xl），悬停时发光效果
- **字体**：标题使用 Noto Sans SC 粗体，正文使用 Noto Sans SC 常规
- **布局风格**：卡片式布局，顶部固定导航栏，响应式网格
- **图标风格**：Lucide 线性图标，与文字搭配使用
- **背景**：深色渐变底色 + 微光粒子动效，营造科技感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区域 | 全屏渐变背景，大标题+副标题，微光粒子动效，向下滚动提示 |
| 首页 | 模块入口 | 四张渐变边框卡片，悬停放大+发光，图标+标题+简介 |
| 首页 | 最新更新 | 横向滚动卡片列表，时间标签 |
| 工具集 | 搜索栏 | 圆角输入框，搜索图标，实时筛选 |
| 工具集 | 分类标签 | 横向滚动标签组，选中态高亮 |
| 工具集 | 工具卡片 | 深色卡片，顶部配图，底部信息区（名称、简介、标签） |
| 工具集 | 详情弹窗 | 模态弹窗，左侧大图，右侧描述+网盘链接+复制按钮 |
| 资源下载站 | 整体布局 | 与工具集页面结构一致，配色微调 |
| 资源下载站 | 详情弹窗 | 与工具集详情弹窗结构一致 |
| 网站导航 | 网站卡片 | 紧凑卡片，网站图标+名称+简介+标签，悬停边框发光 |
| 个人名片 | 个人故事 | 时间线布局，图文交替展示 |
| 个人名片 | 微信入口 | 二维码卡片，点击放大模态 |
| 管理面板 | 表单 | 深色表单，输入框+下拉选择+图片URL输入+富文本描述 |

### 4.3 响应式设计

- 桌面端优先设计（≥1280px）
- 平板适配（768px-1279px）：卡片从4列变2列
- 移动端适配（<768px）：卡片单列，导航栏折叠为汉堡菜单，搜索栏全宽
